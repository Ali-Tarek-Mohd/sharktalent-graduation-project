# SharkTalent Freelance Marketplace

SharkTalent is a graduation project designed as a freelance marketplace platform that connects clients with freelancers through project posting, proposals, freelancer profiles, ratings, and project tracking.

## Project Grade

Graduation Project Grade: **88%**

## Main Features

- Client and freelancer user roles
- Project posting system
- Freelancer proposal system
- Proposal limit to reduce spam
- Rule-based freelancer recommendation concept
- Freelancer profiles and ratings
- Project tracking
- System documentation with UML diagrams and requirements

## Project Purpose

The purpose of SharkTalent is to create a more structured and transparent freelance platform where clients can evaluate freelancers more clearly and freelancers can apply to projects in a controlled way.
